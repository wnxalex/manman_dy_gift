# 抖音直播间礼物点歌机 · 使用说明（网页控制台版）

监听你的抖音直播间，观众送出指定礼物时自动排队播放对应歌曲；空闲时循环播放你设置的背景音乐(BGM)。
所有配置都在**浏览器网页**里点选完成，无需改代码。基于开源项目 [DouYin_Spider](https://github.com/cv-cat/DouYin_Spider)，仅供学习与个人使用。

## 一、装环境（每台电脑只需一次）

1. **Python 3.10~3.12**：https://www.python.org/downloads/ （Windows 安装时勾选 **Add python.exe to PATH**）
2. **Node.js LTS**：https://nodejs.org/zh-cn （签名算法必须用它）

## 二、生成/启动程序

- **打包成可执行文件**：Windows 双击 `build_windows.bat`，Mac 双击 `build_mac.command`。
  完成后在 `dist/` 里得到 `GiftDJ`(.exe)。以后**双击它就能用**，会自动弹出浏览器控制台。
  整个 `dist` 文件夹拷到同系统别的电脑也能直接用（已带 node）。
- **不想打包**：直接双击 `run_windows.bat` / `run_mac.command`，自动装依赖并运行。

启动后浏览器会自动打开 `http://127.0.0.1:5000`（没弹出就手动打开这个地址）。
**那个黑色命令行窗口不要关**，关了程序就停了。

## 三、在网页控制台里操作

1. **① 直播间设置**：填直播间号（网址 `live.douyin.com/` 后面的数字）+ Cookie，点保存。
   - Cookie 获取：Chrome/Edge 打开 https://live.douyin.com 并**登录**（建议小号）→ F12 → 网络(Network) → 刷新 → 点开任一请求 → 请求标头里复制 `cookie:` 后**整串**。
2. **③ 礼物 → 歌曲**：先「上传到歌曲库」传 mp3，再选礼物名 + 歌曲，点「添加映射」。
   - 礼物名可以下拉选常见礼物，也可手输；**启动监听后**，直播间实时收到的礼物会显示成小标签，点一下就自动填进去（最准）。
3. **④ 背景音乐 BGM**：上传多个 mp3，勾选要加入循环的曲子，可开「随机顺序」。空闲时循环播放；一旦收到点歌礼物，BGM 让位给点歌，播完自动继续 BGM。
4. **② 播放设置**：分别调礼物歌音量、BGM 音量、连击窗口（同一人连刷同一礼物只算一次）、队列上限。
5. 点右上角 **▶ 启动监听**。状态灯变绿=已连上，礼物、弹幕会实时滚动在「⑤ 实时日志」。

> 改直播间号 / cookie 后要「停止」再「启动」才生效；礼物映射、音量、BGM 改完点保存即时生效，无需重启。

## 常见问题

| 现象 | 处理 |
|---|---|
| 网页打不开 | 手动浏览器访问 http://127.0.0.1:5000 |
| 提示未检测到 Node.js | 装 Node.js LTS 后重开 |
| 连接失败 | 检查：直播间是否开播、live_id、cookie 是否过期（重新复制） |
| 礼物送了没反应 | 看日志里打印的礼物名，映射里的名字要和它**完全一致**（用实时标签点选最保险） |
| Mac 打开被拦 | 右键 GiftDJ → 打开；或 系统设置→隐私与安全性→仍要打开 |
| Windows 杀毒误报 | PyInstaller 单文件常见误报，加信任 |

## 关于用网易云音乐

见另一份说明的讨论；简单说：**不建议直接远程控制网易云 App**（做不到"某礼物→放某首指定歌"），推荐把想要的歌下载成 mp3 传进歌曲库/ BGM 使用，最稳。

## 风险提示

使用网页逆向接口，属非官方用法，**建议用小号 cookie**，有被风控可能；仅限个人学习研究。
